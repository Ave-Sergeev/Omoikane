### Сборка

Для самостоятельной сборки необходимо наличие установленного инструментария `Rust` (компилятор `rustc` и пакетный менеджер `cargo`).  

1. Установите (если необходимо) инструментарий `Rust`.  
    Для Unix-подобных систем (macOS, Linux, ...) выполните команду в терминале:
    ```shell
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```
2. Выполните проверку правильности установки (в терминале).  
    В случае успешной установки (п. 1), вы увидите примерно это `cargo 1.94.0 ...`.
    ```shell
    cargo --version
    ```
3. Клонируйте проект из `GitHub` и перейдите в корень проекта.  
    Выполните сборку в режиме `release-lto`. После сборки исполняемый файл будет в директории `./target/release-lto/` проекта.  
    ```shell
    cargo build --profile release-lto
    ```
4. Запустите проект через `cargo` (находясь в корне проекта).  

    ```shell
    cargo run --profile release-lto -- [аргументы]
    ```
    Или перенести куда вам удобно скомпилированный бинарный файл, и запустите его из терминала.  

    ```shell
    ~/<path_to_binary_file> [аргументы]
    ```
